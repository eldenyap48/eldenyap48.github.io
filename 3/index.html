Part 2.1: Image "Sharpening"

I first apply a low pass Gaussian blur filter to the image that retains only the low frequencies. I then subtract the result from the original image to get the high frequencies of the image. Finally, I add a scalar multiple (alpha) of the high pass image back to the original image to get the sharpened image. This was also done in a single convolution operation called the unsharp mask filter. The filter is simply (1+alpha) * identity - alpha * gaussian, where the identity filter is simply a filter with 0s everywhere and a 1 in the middle.

When we increase alpha, the sharpening amount, the edges and textures become more pronounced, but halos around strong edges also start to appear. It also increases contrast around edges, and alpha at high values can cause the exaggerated contrast to look unnatural.
